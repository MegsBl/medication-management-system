## About The Project
This project is a medication management system designed to help doctors manage patients' prescriptions and schedules. Doctors can add and track medications for their patients, while patients can view their medication schedule on a calendar. The system aims to improve adherence to medication schedules and streamline communication between doctors and patients.

## Usage
### Features:
1. **Doctor Dashboard**:
   - View a list of patients.
   - Select a patient to add or update medications.
   - Schedule medication times and dosages.

2. **Patient Dashboard**:
   - View a calendar showing when to take each medication.
   - Check detailed information about each prescription.
   - Optional reminders or notifications for medication times.
